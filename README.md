# PSU_navy_2017
1st year EPITECH project

The purpose of the Navy is to reproduce the Naval Battle between two processes using signals.

## Usage
### Serveur
`./navy pos.txt`

### Client
`./navy [pid server] pos.txt`

## Example
Example of pos in the folder `maps`<br />
`./navy maps/pos1`<br />
`./navy 1337 maps/pos2`
